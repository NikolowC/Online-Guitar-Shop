# 🎸 Guitar Shop

A modern for browsing guitar brands, models, and specifications. Built with **React**, **Apollo Client**, and **GraphQL**, the app features filtering and pagination.

---

## 🚀 Features

- View guitar brands and their models
- Filter by guitar type (Electric, Bass, Acoustic)
- View detailed specifications for each model
- Pagination
- Responsive layout

---

## 🔧 Setup Instructions

### 1. **Clone the repository**

```bash
git clone https://github.com/your-username/guitar-shop.git
cd guitar-shop
```

### 2. **Install dependencies**

```bash
npm install
```

### 3. **Set up environment variables**

Create a .env file in the root directory and define the following:

```bash
VITE_GRAPHQL_API_URL=https://your-graphql-api.com/graphql
```

Replace the URL above with your actual GraphQL endpoint.

### 4. **Start the development server**

```bash
npm run dev
```
